
# Next Level Dashboard Improvement: God Mode (OP Dashboard)

To make the dashboard "totally OP", I have implemented a new view mode called **God Mode**.

## Features of God Mode:

1.  **Cyberpunk Aesthetics**:
    -   Dark, high-contrast UI with Neon Cyan and Magenta accents.
    -   Sci-Fi "System Status" widgets.
    -   Monospace fonts for that "hacker" feel.

2.  **Hexagonal Video Hive**:
    -   Instead of a boring valid grid, videos are displayed in a **Hexagon Grid** (Honeycomb).
    -   Hovering over a cell expands it and plays the GIF preview.

3.  **Live Surveillance Wall**:
    -   A side panel features a 2x2 grid of "Security Cams".
    -   These randomly cycle through your video library, giving the impression of a massive surveillance network.

4.  **Real-Time System Vitals**:
    -   **Storage Monitor**: Displays disk usage.
    -   **Network Monitor**: Real-time download speed graph (simulated purely visual for now, but linked to `primaryDownload` speed if active).
    -   **Data Stream**: A scrolling log of system events (imports, errors, status updates) appearing like a terminal feed.

## How to Access:
-   Click the new **DNS Icon** (God Mode) in the top right view switcher (next to the Graph icon).
